Interweb

TCP

TCP staat voor Transmission Control Protocol. Het werkt door een verzoek om te verbinden te sturen. Daarna stuurt de server een erkenning. Waarna de computer weer een erkenning terug
stuurt. Nu is er een verbinding.

definition@source:~$ Wikipedia

UDP

UDP staat voor User Datagram Protocol. Het lijkt een beetje op TCP, maar het heeft niet de connectie stap die TCP wel heeft. UDP stuurt heel simpel gewoon de data door naar het IP dat is opgegeven zonder eerst te proberen te verbinden.

definition@source:~$ Wikipedia

IPv4

IPv4 staat voor Internet Protocol version 4. Het is een protocol om te zorgen dat je met andere computers kan verbinden over een WAN of LAN netwerk.

definition@source:~$ Ome Dennis

IPv6

IPv6 staat voor Internet Protocol version 6. Het is een protocol om te zorgen dat je met andere computers kan verbinden over een WAN of LAN netwerk.

definition@source:~$ Ome Dennis

IP adres

IP adress is het Internet protocol adress. Dit zit er vaak uit als het volgende '127.0.0.1' (IPv4). Het is bedoeld als een soort ID voor als je op het internet bent zodat servers weten waar een verzoek vandaan komt en waar ze een hun antwoord naartoe moeten sturen.

definition@source:~$ Ome Dennis

een subnetwerk is een netwerk dat in twee of meer delen is gedeeld. een subnetmask is er om aan te geven welke pc bij welk netwerk hoort.
